ALS Prolog to Python Interface

The ALS Prolog to Python interface is a Prolog Shared Library which embeds a Python interpreter within ALS Prolog. Prolog can execute Python command and pass basic data type to and from the Python interpreter.

The Interface requires that Python 1.52 or better has been installed on the system.

Example

A simple example of loading the Python interface and calling its predicates. 

?- [python].
yes.

% Set the Python variable a to 2.

?- py_exec('a = 2').
yes.

% Check that the variable a evaluates to 2

?- py_eval('a', X).
X = 2
yes.

% Call the atoi function in the string module to
% convert a hexedicimal string to an integer.

?- py_call(string, atoi('3f0', 16), X).
X = 1008
yes.

Python Interface Predicates

py_exec(command).

Executes the statement command in the Python interpreter.

py_eval(expr, result).

Evaluates the expression expr in the Pyton interpreter and unifies the value of the expression with result.

py_call(mod, func(args..), result).

Calls the function func in the module mod and unifies the result with result.
