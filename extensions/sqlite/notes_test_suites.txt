
All files are at or below: ALSProlog/extensions/sqlite

All can be run with:

alspro <file> -g <Invocation>

All of them load sqlite3_intf.pro, which in turn loads sqlite3_intf.psl
All expect that both sqlite3_intf.pro and sqlite3_intf.psl can be found in
	alsdir/shared
All will create a sqlite3 <Test data file>

The entire collection of tests can be run as a batch by loading and running

	alspro run_tests_examples.pro -g run_tests_examples

Test file		Invocation		Test data file
=========		==========		==============
tests_sqlite3_intf.pro	test_sqlite3		testsq3.db
