ALS Prolog

ALS Prolog is a project of Applied Logic Systems, Inc. It is an open source version of the original ALS Prolog active in the 1980-90s.

The ALS GitHub web page is here and the GitHub organization home for ALS is located here .

☆  ALS Prolog

Gui and tty versions are available for modern Macintosh, Linux, and Windows systems.
